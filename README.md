# Flip Match

[https://dominicyuan1024.github.io/flipmatch/](https://dominicyuan1024.github.io/flipmatch/)  

一款益智小游戏，适合大人陪同小孩一起玩，考验记忆力，帮助小孩记忆词汇
1. 玩家轮流翻开两张卡
2. 成功可继续行动，失败轮到下一位
3. 最终谁收集最多配对谁胜出
